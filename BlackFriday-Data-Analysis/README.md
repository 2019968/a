# 黑色星期五数据分析
基于黑色星期五电商销售数据集，使用 **Python + Tableau** 完成全流程销售数据分析，巩固数据处理、可视化与业务分析能力，为实习求职积累实战项目。

## 技术工具
- Python：Pandas 数据清洗、Matplotlib/Seaborn 探索性分析、特征工程
- Tableau：交互式仪表盘、销售趋势可视化、用户-商品维度分析
- Excel：数据校验、初步统计汇总

## 分析流程
1. 数据预览与清洗：处理缺失值、异常值、重复数据，标准化字段格式
2. 用户画像分析：年龄、职业、消费层级分布统计
3. 商品维度分析：品类销量、销售额、热门商品排行对比
4. 时间维度分析：大促时段销售热度、订单峰值统计
5. 业务关联分析：用户消费行为与商品偏好、销量的关联关系
6. 可视化报告：Tableau 仪表盘搭建、业务结论整理

## 遇到的问题与解决
### 1. 数据缺失与异常值处理不当
初期直接筛选缺失值导致样本量大幅减少，异常值（极端消费金额）误删影响分析结果。
通过分位数界定异常值范围，保留合理极端消费记录；对缺失值采用同分布填充/删除策略，平衡数据完整性与分析准确性，保证后续分析逻辑的可靠性。

### 2. 可视化呈现杂乱，核心重点不突出
初期堆砌过多分析维度，Tableau 仪表盘信息过载，无法快速捕捉关键洞察。
聚焦大促核心分析目标，精简维度（用户画像、商品销量、时间峰值），优化图表配色与布局，突出大促销售高峰、热门商品等核心结论，提升数据呈现的直观性。

## 学习收获
1. 熟练使用 Python 完成数据清洗、分组聚合、特征工程等实战操作，掌握 Pandas 核心语法
2. 系统掌握电商大促数据分析逻辑：用户分层、商品运营、活动效果评估、消费趋势解读
3. 提升 Tableau 仪表盘搭建能力，学会用可视化呈现清晰的数据故事，突出核心洞察
4. 养成严谨的数据分析思维，明确业务需求导向，学会从数据中提炼可落地的分析结论
