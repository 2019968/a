

# 黑色星期五电商销售数据分析
**BlackFriday E-Commerce Sales Data Analysis**

## 项目简介
本项目基于黑色星期五电商销售数据集，通过 **Python + Tableau** 完成从数据清洗、探索性分析、业务洞察到可视化报告的全流程数据分析实战。
面向电商大促场景，输出用户画像、商品销量、销售趋势、消费行为等核心结论，为运营决策提供数据支撑。

## 技术栈
- **数据处理**：Python、Pandas
- **数据分析**：Matplotlib、Seaborn、特征工程
- **数据可视化**：Tableau（交互式仪表盘）
- **辅助工具**：Excel（数据校验、统计汇总）

## 分析目标
1. 挖掘黑色星期五大促期间**用户消费特征**与人群偏好
2. 分析**商品品类销量/销售额**表现，定位爆款与潜力款
3. 识别**销售时段峰值**，辅助活动投放与资源分配
4. 构建可复用的电商大促数据分析流程与可视化模板

## 分析流程
1. **数据预处理**
   - 缺失值、异常值、重复值清洗与处理
   - 字段标准化、数据类型转换、数据去重
2. **探索性数据分析（EDA）**
   - 用户维度：年龄、性别、职业、城市等级、消费能力分布
   - 商品维度：品类销量/销售额排行、交叉偏好分析
   - 时间维度：大促时段销售趋势、订单峰值分布
3. **业务洞察**
   - 用户分层与消费行为关联
   - 商品组合与品类联动分析
   - 大促活动效果评估
4. **可视化输出**
   - Tableau 交互式仪表盘
   - 业务结论与可落地建议

## 核心问题与解决方案
1. **数据质量问题**
   - 问题：缺失值与极端消费值处理不当，导致样本丢失、分析偏差
   - 方案：采用分位数法界定异常值，对缺失值按分布策略填充/删除，平衡数据完整性与分析准确性

2. **可视化信息过载**
   - 问题：图表维度过多，重点不突出，可读性差
   - 方案：精简核心维度，优化图表布局与配色，突出销售峰值、热门品类、高价值用户等关键结论
