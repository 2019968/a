# 黑色星期五销售数据分析

## 项目介绍
基于黑色星期五消费数据，通过 Python 进行数据清洗，并使用 Tableau 完成可视化分析，从用户、商品、城市等维度挖掘消费特征，为营销决策提供参考。

## 技术工具
- Python：pandas, numpy
- Tableau：可视化展示
- GitHub：项目管理

## 分析维度
- 用户年龄与性别消费分布
- 商品类别销售额占比
- 不同城市客单价对比
- 职业与婚姻状况消费差异
- 城市居住年限与消费趋势

## 分析结论
- 26-35 岁男性为核心消费群体。
- 商品类别 1、5、8 贡献主要销售额。
- C 类城市客单价最高，下沉市场消费潜力突出。
- 职业 1、2 消费能力最强，未婚用户消费占比更高。
- C 类城市各居住年限段消费均保持稳定高位。
