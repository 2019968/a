# 淘宝用户行为数据分析项目
基于 SQL + Excel 对2014.11.18-2014.12.18的淘宝用户行为数据进行全流程分析，涵盖数据清洗、行为统计、转化漏斗、RFM用户分层、商品类目分析等核心模块。

## 技术栈
- SQL：数据清洗、多维度指标计算、转化漏斗、RFM模型
- Excel：数据可视化（折线图/条形图/饼图/漏斗图）、用户分层报表
- Git：版本管理与项目发布

## 项目结构
├── sql/ # SQL 分析脚本（按分析流程拆分）│ ├── data_preview.sql # 原始数据预览│ ├── data_cleaning.sql # 数据清洗 + 时间维度拆分│ ├── behavior_analysis.sql # 行为类型统计│ ├── hour_analysis.sql # 小时段行为分析│ ├── week_analysis.sql # 周度行为分析│ ├── funnel_analysis.sql # 多路径转化漏斗│ ├── rfm_analysis.sql # RFM 用户分层│ └── category_analysis.sql # 商品类目转化分析├── excel/ # 可视化分析报表│ └── taobao_user_analysis_report.xlsx # 含 RFM / 类目 / 时间分析图表

### 3. 关键业务结论
- **用户结构**：高价值用户占比超50%，是平台核心营收来源，需重点维护会员权益；
- **召回重点**：流失用户占比33.05%，规模较大，建议推送复购优惠券、个性化推荐进行召回；
- **类目特征**：类目 **2949** 转化率高达25.85%，为全站最高，属于超级爆款类目，可重点加大推广力度；
- **转化瓶颈**：大部分类目转化率在7%-11%之间，普遍低于高转化类目，需优化加购/收藏到购买的转化链路。
